# Imágenes del tutorial

Guarda aquí las imágenes de cada paso del tutorial "Aprende a jugar Online".
No hay que tocar ningún archivo de código — en cuanto el archivo exista con el
nombre correcto, aparece solo dentro de la app.

Nombres esperados (uno por paso, en ese orden):

- paso-1.jpg — Instalar la ISO
- paso-2.jpg — Instalar las Texturas
- paso-3.jpg — Instalar el Savedata
- paso-4.jpg — Configurar ZeroTier
- paso-5.jpg — Configurar PPSSPP
- paso-6.jpg — Buscar Rival
- paso-7.jpg — Colocar la IP
- paso-8.jpg — Crear o Entrar a la Sala

Recomendado: fotos o capturas de pantalla horizontales (relación 16:9),
menos de 1 MB cada una para que la app cargue rápido.

Si quieres usar otro nombre o formato (.png en vez de .jpg), solo cambia el
valor de "image" del paso correspondiente en:
frontend/src/modules/Configuracion/tutorialContent.js
